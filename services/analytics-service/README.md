# analytics-service

## Overview
This is the analytics-service microservice for Chaos World.

## Development

### Prerequisites
- Rust 1.70+
- MongoDB
- Redis

### Running the service
`ash
cargo run --bin analytics-service
`

### Configuration
Configuration files are located in the configs/ directory.

### Testing
`ash
cargo test
`

## API Documentation
See docs/ directory for detailed API documentation.
