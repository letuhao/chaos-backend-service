# user-management

## Overview
This is the user-management microservice for Chaos World.

## Development

### Prerequisites
- Rust 1.70+
- MongoDB
- Redis

### Running the service
`ash
cargo run --bin user-management
`

### Configuration
Configuration files are located in the configs/ directory.

### Testing
`ash
cargo test
`

## API Documentation
See docs/ directory for detailed API documentation.
