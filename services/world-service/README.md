# world-service

## Overview
This is the world-service microservice for Chaos World.

## Development

### Prerequisites
- Rust 1.70+
- MongoDB
- Redis

### Running the service
`ash
cargo run --bin world-service
`

### Configuration
Configuration files are located in the configs/ directory.

### Testing
`ash
cargo test
`

## API Documentation
See docs/ directory for detailed API documentation.
